# diu_2025-2_mvp-pau
Código fuente del Producto Mínimo Viable (MVP) de la Plataforma Unificada de Ayudantías (PAU) para Alumnos. Este MVP usa como base la interfaz entregada por Lovable, a la cual se le implementaron cambios para adaptarse a las metas e hipótesis planteadas, siguiendo los principios de diseño aprendidos en el curso.
