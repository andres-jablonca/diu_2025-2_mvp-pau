Código fuente del Producto Mínimo Viable (MVP) de la Plataforma de Ayudantías Unificada (PAU) para Alumnos. Este MVP usa como base la interfaz entregada por Lovable, a la cual se le implementaron cambios para adaptarse a las metas e hipótesis planteadas, siguiendo los principios de diseño aprendidos en el curso.

## Como ejecutar localmente

```sh
# Instalar dependencias.
npm install

# Levantar server.
npm run dev
```
